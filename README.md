## Installation

    $ gem install irep

## Usage

    # Search and interactive replace text from current directory recursively.
    $ irep before after

    # Search and replace text with some options.
    $ irep OPTIONs before after

    # Display help.
    $ irep --help
    Usage: irep [options]
            --[no-]replace
            --path VAL
